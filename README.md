# Backrest Watcher

Backrest Watcher is a lightweight Windows system tray application that monitors a Backrest log file in the background.

Its job is to detect newly appended log entries that look like warnings or errors and raise a visible tray alert, so issues can be noticed quickly without manually checking the log over and over.

The app is intended as a small companion for Backrest on Windows, providing passive monitoring and faster visibility when something needs attention.
